# JavaScript Assignment 2

This repository contains solutions for **JavaScript Assignment 2** using **HTML, CSS, and JavaScript**.

## 📌 Tasks Implemented

### 1️⃣ Display Even Numbers

A JavaScript program that displays **even numbers from 1 to 100** using a loop.

### 2️⃣ Calculator using Switch

A function that performs basic arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

The program uses a **switch statement** to select the operation.

### 3️⃣ Salary Tax Calculator

A function `findTax()` calculates the tax based on the salary using **JavaScript switch control flow**.

| Salary Range          | Tax Rate |
| --------------------- | -------- |
| 0 – 5,00,000          | 0%       |
| 5,00,001 – 10,00,000  | 10%      |
| 10,00,001 – 15,00,000 | 20%      |
| Above 15,00,000       | 30%      |

### 4️⃣ Sum of Product of Corresponding Digits

A function that calculates the **sum of products of corresponding digits of two numbers**.

Example:

n1 = 6
n2 = 34

Calculation:

(6 × 4) + (0 × 3) = 24
